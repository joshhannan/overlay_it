Overlay It Plugin

Simple overlay for a webpage.

How to Implement

The enclosed css code already has the overlay in a fixed position, so you should most likely activate this on the 'body' tag for best results.

The attributes that Overlay_it supports are as follows:

  • html: 'Place your HTML code here to display in the white window. You can style this however you wish.'
  • close_html: 'Override the default HTML in the close button div. This is if you would want to include an icon, etc.'
	
	$('body').overlay_it({
		html: '<h1/>Testing</h1>',
		close_html: '<span>Close</span>'
	});
